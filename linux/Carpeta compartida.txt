1 - asegúrate de que tienes instalado el paquete cifs-utils. Si no lo tienes, instálalo con:
sudo apt install cifs-utils

2- Crea un directorio donde quieras montar la carpeta compartida
sudo mkdir /mnt/share

3 - Usa el comando mount para montar la carpeta compartida
sudo mount -t cifs //192.168.1.2/compartida /mnt/share

4 - Para ver el contenido, usa:
cd /mnt/share
